# Projeto-Clicker
Este projeto esta sendo realizado em colaboração com a FECAP como projeto interdisciplinar

O projeto inicialmente foi criado com a ideia de ser um clicker, com a historia e tematica de desenvolvedor, ele sera em um estilo parecido ao do cookie clicker, com um sistema de dinheiro, limite de dinheiro e automação


Limite de dinheiro restrito por upgrade 

Upgrades que aumentam o dinheiro gerado por click

Upgrades que aumentam o dinheiro gerado automaticamente 

Todos os upgrades são gerados entorno da tematica discutida, por exemplo computadores melhores, linguagens mais atuais, e etc
